# GM6020 PID 仿真器

这是一个仅针对 `GM6020` 电机的网页仿真器。

建模参数基于 `GM6020` 手册关键数据：

- 额定电压：`24V`
- 转矩常数：`0.741 N·m/A`
- 转速常数：`13.33 rpm/V`
- 转速转矩梯度：`156 rpm/(N·m)`
- 机械时间常数：`3ms`
- 电流给定映射：`-16384 ~ 16384 -> -3A ~ 3A`
